# 🛍️ E-Commerce Sales Analysis Dashboard

A modern and interactive **Power BI Dashboard** designed to analyze e-commerce sales performance and convert raw business data into actionable insights. This dashboard provides a complete overview of sales, customers, products, and orders through interactive visualizations, dynamic KPIs, and drill-down analysis.

## 🚀 Features

* 📈 Executive Sales Overview
* 👥 Customer Analysis
* 📦 Product Performance Analysis
* 🛒 Sales & Order Analysis
* 📊 Dynamic KPIs with Year-over-Year (YoY) Comparison
* 🎯 Interactive Filters & Navigation
* 💡 Dynamic Business Insights
* 🎨 Premium Microsoft/Fortune 500 Inspired UI Design

## 📊 Key KPIs

* 💰 Total Revenue
* 🛍️ Total Orders
* 👤 Total Customers
* 📦 Total Quantity Sold
* 💵 Average Order Value (AOV)
* ⭐ Best Selling Category
* 🏆 Highest Revenue Product

## 🛠️ Tech Stack

* Power BI
* Power Query
* DAX
* Data Modeling

## 🎯 Business Objective

This dashboard enables businesses to monitor sales trends, identify top-performing products, understand customer purchasing behavior, and make data-driven decisions through an intuitive and interactive reporting experience.
